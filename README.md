# HackTheBox Reports

A collection of write-ups and notes from completed **Hack The Box** machines.

This repository documents my methodology and learning process through different challenges, including:

* Reconnaissance
* Enumeration
* Initial Access
* Privilege Escalation
* Post-Exploitation observations

The goal is to build a structured archive of techniques, vulnerabilities, and attack paths encountered during practice.
